# Freelance Quote Calculator

A pricing tool for freelance designers to generate structured, itemised project quotes — instantly.

**Live:** https://vkpillai.github.io/freelance-quote-calculator/

---

## The Problem

Most freelance designers undercharge. Not because they don't know their worth — but because quoting feels awkward, inconsistent, and rushed. Rush fees get forgotten. Usage rights go unmentioned. Complexity never gets factored in.

The result: a number pulled from nowhere, sent with zero confidence.

---

## What This Does

Fill in four fields. Toggle your add-ons. Get a structured estimate with a full breakdown — ready to copy and paste into any proposal or email.

**Inputs:**
- Project type (10 categories — Logo, Brand Identity, Social Pack, Website, Packaging, Pitch Deck, Motion, and more)
- Complexity level (slider — Basic to Complex)
- Revision rounds (1 round to Unlimited)
- Client type (New / Repeat / Referral / Agency)

**Add-ons:**
- Rush Fee (+25%)
- Commercial Usage Rights (+20%)
- Source Files (+15%)
- Brand Strategy Session (+10%)
- Repeat Client Discount (−10%)

**Currency toggle:** ₹ INR · £ GBP · $ USD · € EUR

**Output:** Live estimate with itemised breakdown + one-click copy of a formatted quote block.
